# LangChain 是什么

LangChain 不是大模型本身，它更像是一个 AI 应用开发框架。它的价值在于把很多原本分散的步骤组织起来，例如文档加载、文本切块、embedding、检索、prompt 组装、模型调用和输出解析。

对初学者来说，LangChain 的重点不是“会多少高级 API”，而是先理解它把哪些环节连接了起来。一个最典型的例子就是 RAG：你先读入文档，再切块，再做向量化，再检索，再把检索内容和用户问题一起发给大模型。LangChain 正好把这些步骤都提供了现成组件。

LangChain 本身不负责保存向量，也不负责训练模型。它负责在应用层协调不同服务和模块。你可以把它理解为“AI 应用开发中的积木和胶水”。

在这个教程项目里，LangChain 主要承担四件事：切块、embedding、连接 Chroma、调用聊天模型。只要这四步通了，你对 LangChain 的核心理解就已经建立起来了。
